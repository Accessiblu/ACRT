Accessibility Conformance Reporting Tool (ACRT)

Version 1.0.0

an open source tool
developed by the U.S. Department of Homeland Security
to track testing results from the
Section 508 Conformance Test process

Developed for use in Chrome, Edge and Firefox

For the latest version of the ACRT and support files, all users please visit ACRT GitHub Repository
DHS Employees can also visit ACRT BitBucket Repository

About ACRT

ACRT was developed to record testing results from Testers. ACRT and all data input are stored on the user's local machine. Data are stored as JSON files, which can be further edited for analytics and reporting purposes. Final test results can also be saved as static html files. ACRT is open source - free to use and modify (see the license information for full details). We encourage and welcome feedback and suggestions from the testing community!
You can email us at the DHS Accessibility Help Desk.
