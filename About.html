About ACRT

The Accessibility Conformance Report Tool (ACRT) was developed to record test results from certified Trusted Testers. It replaces the previous Excel spreadsheet used by Trusted Testers.

ACRT was developed to be used in Google Chrome or Microsoft Edge. The tool and all data input into it are stored on the user's local machine.

  • Test data is stored as a JSON file. The stored JSON data can be further manipulated for analytics and reporting purposes.
  • Final test results can be saved as static html file. The results can be shared as needed.

ACRT is open source - free to use and modify. Please see the license information for full details. We encourage and welcome feedback and suggestions from the testing community.
